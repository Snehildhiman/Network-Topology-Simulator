# Network-Topology-Simulator

## Description
A Python-based GUI application that simulates different network topologies and visualizes node connectivity and failures.

## Technologies
- Python
- Tkinter

## Features
- Star, Ring, Mesh, and Bus topologies
- Dynamic node connection visualization
- Node failure simulation
- Network reliability calculation

## How to Run
1. Install Python
2. Run the file:
   python cn.py

## Project Workflow
1. User selects a topology
2. Nodes are created and connected
3. User can click nodes to simulate failure
4. System updates network reliability based on active nodes
